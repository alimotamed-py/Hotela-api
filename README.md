## ✨ About Hotela

Hotela is a modern hotel reservation backend built with Django REST Framework. The project provides a scalable and secure API for hotel booking systems and uses OTP authentication for user login.

## 🚀 Features

- User authentication using OTP
- Hotel management
- Room management
- Reservation system
- RESTful API
- PostgreSQL database
- MIT License

## 🛠 Tech Stack

- Python
- Django
- Django REST Framework
- PostgreSQL
- OTP Authentication

## 📦 Installation

```bash
git clone https://github.com/alimotamed-py/hotela.git
cd hotela

python -m venv venv

# Windows
venv\Scripts\activate

pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## 🔐 Authentication

Authentication is implemented using OTP (One-Time Password).
